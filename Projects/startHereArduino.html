My "Start Here" Arduino Robot

The following is the arduino-based version of the "start here" robot. This project was started after working with the PICAXE platform for several months, and is the start of my transition to ATmega microcontrollers.

The Build

The build is basically the same as the PICAXE-based bot, except that the chassis and attachments are a little more refined. The base is made out of shapelock, with holes drilled out in order to mount the motors. Everything else is attached with either hot glue or the solitary rubber band holding the arduino and protoshield on top of the battery pack.

The Program

The code for this robot was exactly the same in design as the PICAXE bot, except written in wiring. While I originally thought I preferred the BS2 environment, I eventually made the switch to ATmegas and the arduino because the syntax better reflected modern programming langauges.
