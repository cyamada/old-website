Spektrum R/C Robot

This project was assembled using mostly r/c helicopter components, and was the first r/c robot I built.

The Build

The build is basically the same as the PICAXE-based bot. The shapelock chassis and bottlecap as a third wheel basically became the standard build for future projects of the same size. This was also the first robot I created that used servos (modified for continuous rotation) instead of geared motors for the drivetrain. The servos were plugged into the spektrum 2.4 GHz receiver, which was also connected to a battery pack. All of this was then arbitrarily mounted on top of the chassis.

The Control Scheme

There was no coding involved in this project. My Spektrum Dx6i controlled sent out wireless signals to the reciever on the robot. Using the two thumbsticks, I was able to control the motors individually. As the reciever was set up for use with PWM for R/C usage, I was able to control the speed of both wheels.
